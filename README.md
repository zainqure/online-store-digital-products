# Online Store for Digital Products

A simple website where users can browse and download digital learning and productivity tools such as planners, journals, and templates.

## What is this?
This project is a simple online store website for digital products. Users can browse, view, and download items such as study planners, journals, resume templates, and wallpapers. The website includes a homepage, shop page, product detail pages, and a contact form.

## Why does it exist?
This project was created to help students and individuals easily find useful digital tools for studying, productivity, and organization. Many websites are complicated, so this project focuses on a clean and simple design that is easy to use.

## What tools did I use?
- HTML – for structure of the website  
- CSS – for styling and layout  
- JavaScript – for basic interactivity  
- Visual Studio Code – for coding  
- GitHub Pages – for hosting the website  
- Canva – for product images  
- ChatGPT – for learning, debugging, and improving ideas
